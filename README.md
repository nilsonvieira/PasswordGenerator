# PasswordGenerator
App for genereate random password for Linux.

## Homologated On
- Linux Mint 19
- Linux Mint 20
- Debian 9
- Debian 10
- Deepin 15
- Deepin 20

# Instalation
```bash
# Download Zip Pack
wget https://github.com/nilsonvieira/PasswordGenerator/archive/master.zip`
# UnPack
unzip master.zip 
# Enter on directory
cd master
# Execute Build
sudo ./configure
```
